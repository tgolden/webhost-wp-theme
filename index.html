webhost-wp-theme by tgolden

A Wordpress Theme for a webdesign and hosting company.

I am building this theme for a company I work with, and intend to release it to the general public under a Creative Commons License afterwards.

Dependencies

Wordpress 2.7, Grid 960 CSS framework, Thematic wordpress template
